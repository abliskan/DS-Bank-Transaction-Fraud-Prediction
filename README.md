# DS-Bank-Transaction-Fraud-Prediction
## Problem Statement
Banks are constantly under attack from increasingly sophisticated criminal actions in today's financial scene, from intricate identity theft to credit card scammer. Direct financial loss and the decline in consumer trust are the two aspects of the company issue. Every fraudulent transaction that goes unnoticed causes an instantaneous loss of capital as well as possible fines from the authorities. <br>
The severe class disparity in financial data, which ranges from 0.1%, is the technical reality of this issue. There are far more valid transactions (about 99.9% to 0.1%) than fraudulent ones in any particular dataset. The company need a real-time solution that can differentiate between a true security breach and an impulsive vacation purchase without causing the checkout process to lag by more than a few milliseconds.
